01 — WHAT IT DOES

Feedback that points at the real thing

Screenshots go stale and chat threads lose context. annotate.js keeps every note attached to the exact content it’s about.

Highlight & comment

Select the words. Leave the note.

Drag-select any text and a comment popover appears right where you are. Highlights re-anchor with a prefix/suffix match, so they survive edits and reflows instead of drifting off to the wrong paragraph.

Draw · circle · pin · ink

For everything words can’t catch

Box a misaligned button, circle a typo, drop a pin on a single pixel, or sketch freehand over a layout. Each mark carries its own threaded comment and a color, so visual feedback reads at a glance.

Local-first & portable

Your feedback never leaves the page

There’s no server and no database. Comments are saved to localStorage on the reviewer’s own device. To share, export a small JSON file — your teammate imports it and sees every note back in place. No accounts, no tracking.

05 — QUESTIONS

Good things to know

Where are comments stored?

In the reviewer’s own browser via localStorage, namespaced by project and page. Nothing is sent to a server — to share, you export a JSON file and your teammate imports it.

Will highlights break when I edit the page?

Each text annotation stores the exact quote plus a short prefix and suffix. On load it re-finds that text, so notes survive copy edits and layout reflows. If the underlying text is deleted entirely, the comment is kept but shown as “orphaned.”

Does it work with my framework?

Yes — it’s a plain browser script. There are copy-paste recipes for React, Next.js, Vue, Svelte, WordPress, Webflow, Shopify and plain HTML in the README. The rule is the same everywhere: load it once, after the page renders.

Can I show it only to certain people?

Load the script conditionally — e.g. only on a staging domain, or only for logged-in editors. Since it’s one tag, you control exactly when and where it appears.

What does it cost?

Nothing. annotate.js is open source under the MIT license, free for personal and commercial use.
